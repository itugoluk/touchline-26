# Touchline '26

A World Cup 2026 strategy game. Take charge of any of the 48 qualified nations and manage them through the real tournament format — twelve groups, a 32-team knockout, and a final at MetLife Stadium on July 19.

## The game

- **Real 2026 draw.** All twelve groups match the final draw held in Washington D.C. on December 5, 2025, from Mexico–South Africa–South Korea–Czechia in Group A to England–Croatia–Ghana–Panama in Group L. Squads feature each nation's actual key players.
- **Tactical control.** Pick a formation (4-3-3, 4-2-3-1, 4-4-2, 3-5-2, 5-4-1), mentality, pressing line, and playing style. Every choice has a real trade-off in the simulation: high pressing tires after the 70th minute, counter-attacking punishes teams that overcommit, a superior midfield makes possession play stronger.
- **Live matches.** Matches play out minute by minute with a live feed, momentum bar, and scorers weighted by position and quality. You can change your setup at half-time and again around the 70th minute; the opposition manager adjusts too.
- **Full tournament structure.** Top two from each group advance plus the eight best third-placed sides. Knockouts run through extra time and penalty shootouts. If you go out early, the tournament simulates to completion so you know who lifted the trophy.

## Stack

React 19, Vite, Tailwind CSS 4, Framer Motion, Phosphor Icons.

## Run it

```bash
npm install
npm run dev
```
